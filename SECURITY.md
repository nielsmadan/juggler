# Security Policy

## Supported Versions

Juggler is actively developed and only the latest released version receives
security fixes. Please make sure you are on the most recent release before
reporting an issue.

| Version        | Supported          |
| -------------- | ------------------ |
| Latest release | :white_check_mark: |
| Older releases | :x:                |

## Reporting a Vulnerability

Please do **not** open a public issue for security vulnerabilities.

Instead, report them privately through one of these channels:

- **GitHub Security Advisories** (preferred): use the
  [Report a vulnerability](https://github.com/nielsmadan/juggler/security/advisories/new)
  button on the repository's Security tab.
- **Email**: nielsmadan@quantumcraft.io

Please include enough detail to reproduce the issue: affected version, macOS
version, a description of the impact, and step-by-step reproduction where
possible.

## What to Expect

- I aim to acknowledge new reports within a few days.
- Once confirmed, I'll work on a fix and coordinate a release.
- With your permission, I'm happy to credit you in the release notes once the
  fix ships.

Thank you for helping keep Juggler and its users safe.
